Plan de Alimentatie - 2 Luni

Saptamana 1

Lista de cumparaturi:

  • Piept de pui, curcan, somon, cod, ton
  • Oua, iaurt grecesc
  • Avocado, banane, mere
  • Cartofi dulci, cartofi normali, orez brun, quinoa
  • Salata verde, rosii, castraveti, spanac, broccoli, dovlecel
  • Ulei de masline, otet
  • Paine integrala, crackers
  • Seminte de chia, in, migdale, nuci
  • Batoane proteice, linte, naut, fasole rosie

Luni

Pranz: Piept de pui la gratar + cartofi dulci copti + salata cu ulei de masline
Gustare: Iaurt grecesc cu seminte de chia
Cina: Somon + legume la abur + orez brun

Marti

Pranz: Omleta cu legume si avocado + paine integrala
Gustare: Banana + migdale
Cina: Salata cu pui, fasole rosie si paine integrala

Miercuri

Pranz: Quinoa cu legume si curcan
Gustare: Mar + crackers integrali
Cina: Cod cu spanac si cartofi la cuptor

Joi

Pranz: Salata cu ton, naut si legume
Gustare: Iaurt cu seminte de in
Cina: Curcan la gratar + broccoli si orez brun

Vineri

Pranz: Paste integrale cu legume si pui
Gustare: Batoane proteice
Cina: Salata calda cu legume coapte si somon

Sambata

Pranz: Orez cu linte si legume
Gustare: Fructe uscate si nuci
Cina: Omleta cu legume si salata verde

Duminica

Pranz: Friptura de vita slaba cu cartofi natur si salata
Gustare: Iaurt grecesc + banana
Cina: Supa crema de legume + crackers integrali

Saptamana 2

Lista de cumparaturi:

  • File de pastrav, muschi de vita, carne de curcan tocata
  • Oua, branza cottage, kefir
  • Pere, prune, fructe de padure
  • Orez basmati, couscous, hrisca
  • Ardei gras, morcovi, conopida, sparanghel, varza rosie
  • Lamaie, usturoi, patrunjel
  • Paine de secara, tortilla integrala
  • Fistic, caju, seminte floarea-soarelui
  • Linte rosie, mazare, ciuperci

Luni

Pranz: Muschi de vita cu hrisca si ardei gras
Gustare: Kefir cu fistic
Cina: File de pastrav cu conopida si morcovi la abur

Marti

Pranz: Omleta cu ciuperci si paine de secara
Gustare: Piersica si cateva nuci caju
Cina: Curcan tocat cu sos de rosii si couscous

Miercuri

Pranz: Salata calda cu mazare, sparanghel si ou fiert
Gustare: Branza cottage cu pere
Cina: Supa de linte rosie cu lamaie si patrunjel

Joi

Pranz: Tortilla integrala cu carne tocata si legume
Gustare: Kefir cu seminte de floarea-soarelui
Cina: File de pastrav cu legume la cuptor

Vineri

Pranz: Hrisca cu ciuperci si salata de varza rosie
Gustare: Fructe de padure cu iaurt
Cina: Tocanita de curcan cu morcovi si orez basmati

Sambata

Pranz: Couscous cu legume si naut
Gustare: Prune + caju
Cina: Omleta cu sparanghel si rosii

Duminica

Pranz: Friptura de vita cu cartofi natur si varza rosie
Gustare: Branza cottage cu fructe
Cina: Supa de mazare cu crutoane de secara

Saptamana 3

Lista de cumparaturi:

  • Carne de pui, peste alb, oua, tofu
  • Iaurt natural, lapte vegetal
  • Kiwi, mango, mere
  • Cartofi dulci, fulgi de ovaz, cuscus integral
  • Broccoli, rosii cherry, spanac, castraveti, ciuperci
  • Mustar, ulei de masline
  • Paine integrala, tortilla
  • Seminte dovleac, nuci braziliene
  • Linete, naut, fasole alba

Luni

Pranz: Tofu la gratar cu broccoli si cuscus
Gustare: Kiwi + seminte de dovleac
Cina: Peste alb cu piure de cartofi dulci si castraveti

Marti

Pranz: Salata cu ou, spanac si ciuperci sotate
Gustare: Iaurt cu mango
Cina: Tacos cu pui si legume in tortilla integrala

Miercuri

Pranz: Cuscus cu naut si rosii cherry
Gustare: Mar + nuci braziliene
Cina: Supa de legume cu paine integrala

Joi

Pranz: Piept de pui la tigaie + salata cu spanac si mustar
Gustare: Fulgii de ovaz cu lapte vegetal si kiwi
Cina: Tofu cu legume coapte si orez brun

Vineri

Pranz: Quiche cu ciuperci si broccoli
Gustare: Iaurt + seminte
Cina: File de peste alb cu linte si rosii cherry

Sambata

Pranz: Salata cu tofu, mango si mix de seminte
Gustare: Fructe uscate + nuci
Cina: Omleta cu legume si castraveti

Duminica

Pranz: Pui cu cartofi dulci si salata
Gustare: Iaurt + mar
Cina: Supa crema de linte cu crutoane
